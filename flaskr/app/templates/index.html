Bachelor Thesis

DATX11-DIT561-VT25-93

Countless individuals rely on authentication systems to safeguard their personal and professional lives, from accessing secure accounts to protecting sensitive data. Yet, traditional password-based systems remain vulnerable to breaches, phishing attacks, and weak password practices, leaving users at risk. Addressing this critical security challenge, this project is set on developing a user-friendly face recognition authentication system

Proof of Concept
A basic functional version to test the feasibility of facial authentication in real-world scenarios
Open-Source
The full project code is publicly available, encouraging contributions
